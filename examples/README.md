# Example profiles

These are finished **AI Interaction Profiles** — exactly what the builder produces
when you pick each archetype and leave the defaults. Use them as ready-made
starting points or as "here's what good looks like" references.

Paste any of these into ChatGPT (Custom Instructions), Claude (Profile or Project
instructions), Gemini, Copilot, or a generic system-prompt box.

| File | Archetype | Best for |
|---|---|---|
| [secretary.md](secretary.md) | The Organizer | Planning, tracking, staying on top of tasks |
| [companion.md](companion.md) | The Companion | Thinking out loud with a warm, patient presence |
| [coach.md](coach.md) | The Motivator | Momentum, encouragement, clear next steps |
| [devils-advocate.md](devils-advocate.md) | The Challenger | Stress-testing ideas before the real world does |
| [creative-partner.md](creative-partner.md) | The Creator | Riffing and building creative work together |

Want a different mix? Open the [builder](https://angrymunky.github.io/build-your-ai/)
and tweak the sliders — these files were generated the same way.
